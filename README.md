# React Developer Profile Card

## Overview

This project is a simple React application that displays a personal developer profile card. It showcases a developer's profile information, technical skills, and proficiency levels using reusable React components.

## Live Demo

**Project URL:** https://react-challenge-shembio.netlify.app/

## Features

* Developer profile card with avatar
* Professional introduction section
* Dynamic skills list rendered using React's `map()` function
* Skill proficiency indicators with emojis
* Color-coded skill tags
* Reusable React components
* Responsive and clean user interface

## Technologies Used

* React
* JavaScript (ES6+)
* HTML5
* CSS3
* ReactDOM

## Project Structure

### Components

#### App Component

The root component that organizes the overall layout of the profile card.

#### Avatar Component

Displays the developer's profile image.

#### Intro Component

Contains the developer's name and professional summary.

#### SkillList Component

Dynamically renders all skills from the skills array.

#### Skill Component

Displays individual skills with:

* Skill name
* Background color
* Proficiency level indicator

## Installation

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project directory:

```bash
cd react-profile-card
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm start
```
